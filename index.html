Selectores CSS

Los selectores CSS son fundamentales para aplicar estilos a los elementos HTML en una página web. A continuación, te presento los conceptos clave relacionados con los selectores CSS:

  • Descripción: Aplica estilos a todos los elementos de un tipo especifico

    Sintaxis:

    Elemento { propiedad: valor; }

    Ejemplo:

    p {

    color: blue; }

  • Descripción: Aplica estilos a todos los elementos con una clase específica. Se usa un punto (.) antes del nombre de la clase.

    Sintaxis:

    .clase { propiedad: valor; }

    Ejemplo:

    .highlight {

    color: blue; }

  • Descripción: Aplica estilos a un único elemento con un ID específico. Se usa un signo de número (#) antes del nombre del ID.

    Sintaxis:

    #id { propiedad: valor; }

    Ejemplo:

    #header {

    font-size: 24px; }

  • Descripción: Aplica estilos a elementos que tienen un atributo específico o un valor de atributo.

    Sintaxis:

    Atributo { propiedad: valor; }

    Ejemplo:

    input[type='text'] {

    border: 1px solid black; }

  • Descripción: Aplica estilos a elementos que son descendientes de otro elemento.

    Sintaxis:

    padre descendiente { propiedad: valor; }

    Ejemplo:

    input[type='text'] {

    border: 1px solid black; }

  • Descripción: Aplica estilos a elementos que son hijos directos de otro elemento.

    Sintaxis:

    padre > hijo { propiedad: valor; }

    Ejemplo:

    ul > li {

    list.style.type: none; }

  • Descripción: Aplica estilos al primer elemento que sigue a otro elemento, que es su hermano adyacente.

    Sintaxis:

    elemento + elemento { propiedad: valor; }

    Ejemplo:

    h2 + p {

    margin-top: none; }

  • Descripción: Aplica estilos a todos los elementos que son hermanos del elemento especificado.

    Sintaxis:

    elemento ~ elemento { propiedad: valor; }

    Ejemplo:

    h2 ~ p {

    color: none; }

  • Descripción: Aplica estilos a todos los elementos en la página.

    Sintaxis:

    * { propiedad: valor; }

    Ejemplo:

    * {

    box-sizing: border-box; }

  • Descripción: Aplica estilos en función del estado del elemento.

    Sintaxis:

    elemento:peseudo-clase { propiedad: valor; }

    Ejemplo:

    a:hover {

    color: red; }

  • Descripción: Aplica estilos a partes específicas de un elemento.

    Sintaxis:

    elemento:peseudo-elemento { propiedad: valor; }

    Ejemplo:

    p::first-line {

    font-weight: bold; }
